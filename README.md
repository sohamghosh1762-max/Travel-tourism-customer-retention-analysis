🏨 Project Overview

This project is a professional hospitality analytics and predictive intelligence platform designed to analyze hotel booking behavior, customer cancellation patterns, and dynamic pricing trends using real-world data analytics methodologies.

The system focuses on uncovering high-value business insights that help hospitality businesses improve customer retention, reduce booking cancellations, and optimize Revenue Per Available Room (RevPAR).

The project simulates a real-world analytics pipeline used by modern hotel chains, travel agencies, and revenue management teams.

📈 Key Business Insights

The analysis identifies several major drivers of booking cancellations and revenue fluctuations:

Long lead times increase cancellation probability
Non-refundable deposits reduce churn
Seasonal demand significantly impacts ADR pricing
Customer behavior patterns vary across market segments
Pricing elasticity changes during peak tourism seasons

These insights support strategic decision-making for hospitality revenue management teams.
